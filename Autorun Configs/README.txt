NeuroMod Toolbox; Autorun_Configs Folder README
Author: Tony de Schultz
______________________________________________

This folder contains all autorun config files the user can select and modify in the autorun manager window. 

Files in the Config_Files folder contain the currently active Configs, which can be modified and selected in the autorun window. Don't change the folder name or otherwise the autorun config window might not work properly anymore
