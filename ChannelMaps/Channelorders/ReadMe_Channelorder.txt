If you recording data is saved as one file for each channel, those files are read in the order they are saved in the folder (sortet by names). The first file is channel 1 and so on. 
If the channel order of your probe is different, you can use the "Load Channel Order" button in the Extract Data window and select a .mat file containing the true channel order of your probe. If the first channel loaded based on file names is truly channel 10 of your probe, the first value of the vector in this .mat file has to be a 10. This way you indicate, at which true position the loaded files are.
