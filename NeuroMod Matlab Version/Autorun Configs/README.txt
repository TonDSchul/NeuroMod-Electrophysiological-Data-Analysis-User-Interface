NeuroMod Toolbox; Autorun_Configs Folder README
Author: Tony de Schultz
______________________________________________

This folder contains all autorun config files the user can select and modify in the autorun manager window to change the automatic analysis over multiple folder. 

MATLAB files in the 'Config_Files' folder contain the currently active Config files, which can be modified and selected in the autorun window. Don't change the folder name or otherwise the autorun config window might not detect them properly. Config files can be deleted and added as wished and names can be changed as needed as well.
